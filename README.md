Σύστημα Διαχείρισης Ταινιών & Σειρών (PyFlix)


1. Περιγραφή:

Η εφαρμογή υλοποιεί ένα σύστημα καταλόγου ταινιών και τηλεοπτικών σειρών. Ο βασικός σκοπός του συστήματος είναι η διαχείριση και η αναζήτηση οπτικοακουστικών έργων και των συντελεστών τους.

Οι βασικοί χρήστες είναι δύο:

    Απλός Χρήστης: Μπορεί να αναζητήσει ένα έργο με βάση τον τίτλο του και να δει πληροφορίες (ημερομηνία, βαθμολογία, περίληψη) καθώς και τους συντελεστές (Σκηνοθέτες, Ηθοποιοί).

    Διαχειριστής (Admin): Κάνει είσοδο στο σύστημα (login) και έχει τη δυνατότητα να καταχωρεί νέα έργα, να καταχωρεί νέους καλλιτέχνες και να συνδέει τους καλλιτέχνες με τα έργα.

8. Γνωστά Περιοριστικά Σημεία / Μελλοντική Επέκταση

    Διατήρηση Δεδομένων: Προς το παρόν, η αποθήκευση γίνεται στη μνημη. Κάθε φορά που τερματίζεται η εκτέλεση του προγράμματος, τα δεδομένα που προσέθεσε ο διαχειριστής χάνονται. Μια μελλοντική επέκταση θα ήταν η αποθήκευση δεδομένων σε μια βάση.

    Έλεγχος Τύπων: Το σύστημα λαμβάνει όλες τις τιμές ως κειμενο. Δεν υπάρχει αυστηρός έλεγχος  για τον τύπο των ημερομηνιών, οπότε αν ο χρήστης εισάγει μη έγκυρη μορφή, το σύστημα την αποθηκεύει ως απλό κείμενο. Μελλοντικά θα μπορούσε να προστεθεί ένα στάδιο όπου τα inputs ελέγχονται.
